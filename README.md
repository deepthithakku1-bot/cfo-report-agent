# CFO Report Generator

This project automates weekly financial reporting and generates executive-level insights.

## What it does
- Calculates revenue vs budget variance
- Identifies cost drivers
- Generates executive summary using AI

## Tools used
- Zapier
- Google Sheets
- AI prompting

## Sample Output

The variance indicates that weekdays struggle against projections while weekends show strong performance, especially on Sunday.
2. **Margin Trends**: The EBITDA margin seems stable with fluctuations aligning closely to revenue. Despite some lower revenues, EBITDA showed resilience, increasing on higher revenue days like Thursday and Sunday, illustrating effective cost management.

4. **Worst Performance**: Monday (-$20), Wednesday (-$10) compared to budget. \n
